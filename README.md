# QuizGame
Course Project for Problem Solving and Programming (PSAP). 
It uses the concepts of File Handling for the storage and retreival of High Scores.
It allows the user to choose a domain out of the following six domains:
- Science and Technology
- History
- Entertainment
- Current Afairs
- Sports 
- Wildlife

The user is first asked to enter his/her name and then asked to choose anyone of the subjects to play the quiz. 
Each subject consists of 10 questions and has four options. For every correct answer, the user scores 10 points. 
The quiz has no negative marking, which means that for an incorrect answer the user will score +0 marks.
